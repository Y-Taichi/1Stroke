# ローカルで実行する方法

**前提条件:** Node.js

1. 依存関係をインストールします:
   `npm install`
2. 環境ファイル（例: `.env.local`）に API キーを設定します。
3. アプリを実行します:
   `npm run dev`

# プロジェクトについて

このプロジェクトは一筆描き（single-stroke）の練習アプリです。
画像をトレースし、記憶から描き、AI による解析（頂点マッチング）でフィードバックを得られます。

繰り返し遊ぶことで描画の精度が自然に向上します。

このコードは Gemini と the-pen によって書かれました。
